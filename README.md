## 🌍 About The Project

EcoTrack is a fully-featured desktop web platform built to empower users in the fight against climate change. It calculates carbon emissions generated from daily activities like **transportation, energy consumption, diet, shopping,** and **travel**, providing an intuitive breakdown of an individual's ecological footprint.

Built over the course of **Hack Horizon 2.0**, organized at **Arka JAIN University**, the final project features a premium glassmorphic UI, dynamic client-side routing, advanced localized state management, and real-time PDF report generation.

---

## ✨ Key Features

- **📊 Comprehensive Calculators:** Assess transportation, diet, and energy metrics to intelligently estimate your daily and yearly `kg CO₂` impact.
- **💡 Smart Dashboard:** A seamlessly integrated user dashboard serving as the control center for exploring carbon solutions and calculators.
- **🌎 UN Climate Solutions Interface:** Native hub displaying core verifiable climate strategies directly citing United Nations methodology.
- **📄 Downloadable PDF Reports:** Intelligently capture emission analytics via layered DOM processing to generate high-quality analytical PDF exports directly in the browser.
- **🎨 Premium 'Glassmorphism' UI:** Styled from scratch with Vanilla CSS featuring rich, dark-themed variables, dynamic transitions, and modern blur filters.

---

## 🛠️ Built With

- **HTML5:** Crisp semantic architecture.
- **CSS3:** Constructed using CSS Custom Properties, layout grids, and hardware-accelerated animations natively without bulky external packages like Bootstrap or Tailwind.
- **Vanilla JavaScript:** Engineered with a custom lightweight client-side router and view controller handling complex modal interactions and states without React.
- **Third-Party Integrations:**
  - [Lucide Icons](https://lucide.dev/) for crisp scalable SVGs.
  - [jsPDF](https://github.com/parallax/jsPDF) & [html2canvas](https://html2canvas.hertzen.com/) working in tandem via a segmented-tile approach for perfect cross-browser PDF rendering.
